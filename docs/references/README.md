# 外部参考文档

存放外部依赖的 LLM 友好版本。Agent 无法访问外部网站，需要将关键参考文档本地化。

## 使用指南

当项目依赖的外部库文档对 agent 工作很重要时：
1. 提取关键 API 说明（不是全文复制）
2. 保存为 markdown 到此目录
3. 在对应模块的 CLAUDE.md 中引用
